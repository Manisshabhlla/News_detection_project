# Fake News Detection AI 🔎📰

This project uses **Natural Language Processing (NLP)** to detect fake news.  
Built using **Python, Flask, and Machine Learning**, this AI helps verify news accuracy.  

## 🚀 How to Use  
1️⃣ Install dependencies: `pip install -r requirements.txt`  
2️⃣ Run Flask server: `python app.py`  
3️⃣ Send a POST request with news text: `http://127.0.0.1:5000/predict`  

## 📂 Project Structure  
- `app.py` → Flask web app  
- `fake_news_model.pkl` → Trained AI model  
- `tfidf_vectorizer.pkl` → Text processing  
- `predict.py` → Handles predictions
